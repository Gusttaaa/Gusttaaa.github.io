  • ChatBot Imobiliária

    Este projeto é uma solução de automação ponta-a-ponta construída com N8N. Utilizando um agente de IA, o chatbot acessa um banco de dados para realizar consultas dinâmicas sobre imóveis e, através da integração com APIs, executa um fluxo completo de agendamento. A automação inclui a criação de eventos no Google Agenda e o envio de mensagens de confirmação via WhatsApp, demonstrando a capacidade de orquestrar diferentes sistemas para criar uma aplicação funcional e robusta.
